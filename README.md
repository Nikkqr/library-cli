# library-cli

### Library Management System
Консольное приложение для управления каталогом книг в библиотеке. Выполнено в качестве тестового задания.


### Описание принятых решений
Приложение построено на трёхслойной архитектуре:
- presentation (консольный ввод/вывод)
- application (бизнес-логика)
- data (хранение в памяти).

Уникальный ID генерируется в репозитории. Поиск работает регистронезависимо с частичным совпадением одновременно по 
названию и автору. Сервис возвращает готовые строки для вывода, что позволяет легко тестировать
бизнес-логику с моками без привязки к консоли.  

Реализованы Unit-тесты, покрывающие следующие сценарии:
- Валидация при добавлении: проверка корректности ответа при попытке завести дубликат книги.
- Удаление: проверка успешного удаления существующей книги и корректной обработки ситуации, когда книги с таким ID нет в хранилище.
### Технологии
* Language: Java 21
* Build System: Maven
* Testing: JUnit 5, Mockito

### Инструкция по запуску
```
#клонирование репозитория
git clone https://github.com/Nikkqr/library-cli
cd library-cli

#компиляция
mvn compile

#запуск тестов
mvn test

#запуск приложения
mvn exec:java "-Dexec.mainClass=com.library.Main"
```
