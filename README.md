# library-cli

### Library Management System
Консольное приложение для управления каталогом книг в библиотеке. Выполнено в качестве тестового задания.


### Описание принятых решений
Приложение построено на трёхслойной архитектуре: presentation (консольный ввод/вывод), 
application (бизнес-логика) и data (хранение в памяти). ID генерируется в репозитории. Поиск работает регистронезависимо с частичным совпадением одновременно по 
названию и автору. Сервис возвращает готовые строки для вывода, что позволяет легко тестировать
бизнес-логику с моками без привязки к консоли.

### Технологии
* Language: Java 21
* Build System: Maven
* Testing: JUnit 5, Mockito
